This demo project shows how Vercel's [`redirect` or `rewrite` functionality](https://vercel.com/docs/project-configuration#file-based-configuration) can be used to forward from one domain to another. The whole project content of vercel-forwarding-test.vercel.app is [this config file](https://github.com/gh-stats-extended/vercel-forward-test/blob/main/vercel.json). The stats card shown below are actually generated by github-stats-extended.vercel.app .
